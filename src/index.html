Astheno makes it easy to run quality, data-conscious services all on your own hardware.

Own your data

All Astheno apps are self-hostable and are transparent about only storing the data they need to.

Humans first

Software is about people. The ease of use and mix of intuitiveness and customization should reflect that.

Machines second

Fully featured APIs are available to use and/or ingest actions from each application, while maintaing the other principles first.

Independent

Designed to have as few dependencies as possible, there's no feature limitation for only running one app in the collection.

Integrated

However if you choose to operate many, they work tightly together to have an even more seamless user experience.

Modular

Astheno is not the only open source project out there, and integrations are made with many popular apps from the greater ecosystem.

Open source

All the code is managed out in the open on GitHub and bug reports and feature requests are always welcome.

Built for longevity

Bringing together simplicity and ease of use, applications will be maintained and supported for as long as you use them.

Secure

Data is dangerous and fine-grained controls are offered so read and write access to objects is only given to who you approve.

Lightweight

Built as single binaries, Astheno apps are portable and only a few MB in size to get up and running in no time.
